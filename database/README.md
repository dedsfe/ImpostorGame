# Database

Esta pasta concentra o desenho do banco de dados da Noite de Jogos.

## Objetivo

Dar suporte a:

- catálogo centralizado de conteúdo
- categorias e dificuldades
- futuras sessões de jogo
- expansão sem depender de hardcode no frontend

## Arquivos

- `schema.sql`
  - schema relacional inicial
- `migrations/`
  - reservado para migrações futuras
